# 友链申请指南 🔗

欢迎来到友链申请页面！如果你也有自己的博客或个人网站，想要和我互换友链，可以按照下面的步骤提交申请。

---

## 申请方式

我通过 **GitHub Issues** 来管理友链申请，这样方便追踪进度，也更加透明。

### 📝 申请步骤

1. 点击下方按钮，前往我的 GitHub 仓库提交一个 Issue

   👉 **[在 GitHub 上提交友链申请](https://github.com/Mufumoca/blog/issues/new?assignees=&labels=friend-link&template=friend-link.yml&title=%5BNew+Friend%5D+)**

2. 按照 Issue 模板填写以下信息：

   - **网站名称**：你希望显示为友链标题的文字
   - **网站链接**：你的网站地址（请确保是 HTTPS）
   - **网站介绍**：一句话介绍你的网站
   - **头像**：你的网站头像 URL（1:1 效果最佳，建议使用图片直链）

3. 提交后我会尽快审核，通过后会将你的链接添加到我的 [友链页面](/friends/)

---

## 申请要求

为了保证友链质量，希望你的网站满足以下条件：

- ✅ 网站可以正常访问
- ✅ 有实质性的内容（不是空白站点）
- ✅ 没有违规或不良内容
- ✅ 建议已将本站添加到你的友链中（互链优先）

## 常见问题

### 提交后多久会处理？

通常会在 **1-3 天** 内处理，具体取决于我的空闲时间。

### 如果我的申请被拒绝了怎么办？

我会在 Issue 中说明原因，你可以修改后重新提交。

### 网站改版后需要更新友链信息怎么办？

在原来的 Issue 中留言即可，我会及时更新。
